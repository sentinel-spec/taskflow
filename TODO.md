# Roadmap & TODO List (Sensata API)

## ✅ Завершено (MVP + Agile + Collaboration)
*   [x] **Core:** Workspace, Project, User, Auth.
*   [x] **Issues:** CRUD, Sequences, States (Kanban).
*   [x] **Agile:** Cycles (Sprints), Modules (Epics), Estimates.
*   [x] **Collaboration:** Activity Log, Notifications, Favorites.
*   [x] **Knowledge:** Pages, Issue Drafts.
*   [x] **Integrations:** Webhooks, Import/Export (JSON), Basic Analytics.

## 🚀 Priority 6: Instance Administration (SuperAdmin)
*   [ ] **Instance Users:** Просмотр и управление всеми пользователями системы.
*   [ ] **Instance Workspaces:** Управление всеми воркспейсами.
*   [ ] **Instance Config:** API для динамической настройки SMTP, AI и лимитов (вместо .env).
*   [ ] **Registration Control:** Управление открытой регистрацией и приглашениями на уровне сервера.

## 💬 Priority 7: Issue Deep Dive (Enrichment)
*   [ ] **Issue Comments:** Система комментариев к задачам.
*   [ ] **Issue Attachments:** Загрузка файлов напрямую к задачам.
*   [ ] **Issue Relations:** Связи между задачами (Дубликат, Блокирует, Зависит от).
*   [ ] **Issue Links:** Внешние ссылки для задач.
*   [ ] **Issue Reactions:** Эмодзи-реакции на комментарии и задачи.

## 🛠 Priority 8: Advanced Workspace & Project Tools
*   [x] **Labels:** Глобальная система тегов (Labels) для задач.
*   [x] **Views:** Сохраненные фильтры и кастомные представления (Views).
*   [x] **Intake (Inbox):** Входящие заявки для сбора фидбека/багов от внешних пользователей.
*   [x] **Project Members Roles:** Гибкая настройка прав внутри проекта (Viewer, Editor, Admin).

## 🔍 Priority 9: Search & Discovery
*   [x] **Global Search:** Поиск по всей системе (задачи, проекты, страницы).
*   [x] **Recent Visits:** Отслеживание последних посещенных объектов для быстрого перехода.

## 🔑 Priority 10: Infrastructure & Security
*   [x] **API Tokens:** Генерация токенов доступа для сторонних скриптов (PAT).
*   [ ] **MFA / OAuth:** Расширение системы аутентификации (Google, GitHub login).
